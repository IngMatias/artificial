Excel offers real-time data visualization and robust computational capabilities. While it can be a valuable tool for many data-related tasks, it should not be deemed obligatory for a proficient data analyst. Data analysts should have the flexibility to choose from a range of specialized data analysis and visualization tools based on the specific requirements of their projects. Excel can be a useful addition to their toolkit, but it should not limit their ability to explore more powerful and tailored solutions for in-depth data analysis.

Cells

An Excel cell is where a row and column intersect, representing the smallest unit of data in a spreadsheet. It serves as the fundamental building block for organizing, storing, and manipulating data within a worksheet. Cells can be thought of as data containers that can hold various types of information, such as text, numbers, formulas, and even images.

Relative Reference

In Excel, you can reference other cells by using the equal sign (=) and specifying the column letter followed by the row number. This referencing method is commonly referred to as 'relative cell referencing.' When you use relative references, it means that the formula or function you create will adjust automatically when copied to other cells.

Extend

One of the most powerful capabilities of Excel is its ability to extend formulas. Excel's formula extending functionality allows you to apply a single formula to multiple cells, automatically adjusting the cell references in a relative manner.

  1. Move your mouse cursor to the bottom-right corner of the cell containing the formula. When you hover over this area, your cursor will change to a black plus sign.
  2. Click and hold the left mouse button while keeping it pressed.
  3. While still holding the mouse button, move your cursor to the desired cells where you want the formula to be extended.
  4. Release the mouse button.

If you reference an empty cell, it will be interpreted as a '0' value.

Pay attention to how when you extend a formula from one cell to another, the references in the formula will shift by one cell as well.

Absolute Reference

To prevent the previously mentioned issue of unintentional shifting when extending formulas, you can utilize absolute references. Unlike relative references, which adjust when formulas are extended, absolute references remain fixed. To create an absolute reference, you begin with the equal sign (=), followed by a dollar sign ($) before specifying the column letter and a second dollar sign ($) before the row number.

Operators

Every calculation in Excel begins with the universally recognized equal sign (=), which is followed by various combinations of operands and operations.

  • + Addition
  • - Subtraction or Negation
  • * Multiplication
  • / Division
  • % Percent
  • ^ Exponentiation
  • = Equal to
  • > Greater than
  • < Less than
  • >= Greater than or equal to
  • <= Less than or equal to
  • <> Not equal to
  • & concatenates two values
  • : reference to all the cells between two references(range)
  • , combines multiple references into one reference(union)
  • (space) produces one reference to cells common to the two references(intersection)

Functions

As you're already aware, any calculation in Excel initiates with the equals sign. In this section, we will introduce you to simple yet highly useful functions. Functions are predefined behaviors designed to streamline and simplify common tasks, helping you save time and enhance efficiency in your spreadsheet work.

The various inputs of a function in Excel are separated by semicolons (';').

SUM

Returns the sum of the values of the cell references.

Syntax: =SUM(number1, [number2], ...)

SUMPRODUCT

Useful for calculating the total of a supermarket receipt where you have price per quantity and quantity, and you want to get the sum of each price multiplied by quantity.

Syntax: =SUMPRODUCT(number1, [number2], ...)

COUNTIF

Returns the quantity of cells that meet a specified criterion. The criterion is defined within double quotation marks.

=COUNTIF(range, ">3") counts cells with values greater than 3.
=COUNTIF(range, "tomatoes") counts cells where "tomatoes" appears.
=COUNTIF(range, "<>"&B4) counts cells with values not equal to the value in cell B4.

IF

Returns a true value when a specified condition is true and a false value when the condition is false.

Syntax: =IF(condition, trueValue, falseValue)

RAND

Returns a random number between 0 (inclusive) and 1 (exclusive).

=RAND()*100 returns a random number between 0 and 100

RANDBETWEEN

Returns a random integer between specified lower and upper limits.

Syntax: =RANDBETWEEN(lowerLimit, upperLimit)

RANDARRAY

Returns an array of random numbers within specified limits.

Syntax: =RANDARRAY(numOfRows, numOfColumns, min, max, integer)

INT

Returns the whole number part of a value.

Syntax: =INT(value)

MIN and MAX

Returns the minimum and maximum values in a set of numbers, respectively.

Syntax: =MIN(number1, [number2], ...) and =MAX(number1, [number2], ...)

AVERAGE and STDEV

Calculate the average and standard deviation of a range of values, respectively.

Syntax: =AVERAGE(range of values) and =STDEV(range of values)

Z-score Normalization

Standardizes a value using the z-score formula: Standardized value = (actualValue - mean) / standard_dev.

Syntax: STANDARDIZE(num, mean, standard_dev)

Min-Max Scaling

Scales a value to a range between 0 and 1 using the formula: x = (x - min) / (max - min).

Excel has no formula for this calculation.
